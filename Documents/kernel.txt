kernel functions at INT 20h

------------------------------------------------------------
AH = 00h	start process

returns:
AL: result code

------------------------------------------------------------
AH = 01h	kill process

CX:DX		process name

returns:
AL: result code

------------------------------------------------------------
AH = 02h	list first process

CX:DX		pointer to data structure

returns:
AL: result code

------------------------------------------------------------
AH = 03h	list next process

CX:DX		pointer to data structure

returns:
AL: result code


other
------------------------------------------------------------
INT 80h	Sleep (finish scheduler time slice)
------------------------------------------------------------
INT 81h	Terminate current process
