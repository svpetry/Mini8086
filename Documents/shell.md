# Shell commands

|||
|---|---|
| cls | clear screen |
| ver | show version information |
| help | show help |
|||
| dir | list directory content |
| ren | rename file |
| del | delete file |
| cd | change directory |
|||
| time | show/set time |
| date | show/set date |
|||
| reboot | reboot system |
|||
| ps | show processes |
| kill | kill process |
|||
| type | display file contents |
| showpic | show picture |
